# Marvel vs DC

## Description

Marvel vs DC is a game where the player plays as a Marvel super-hero(Spider-man) and tries to avoid DC enemies, by going up and down, left and right.

## MVP (DOM - CANVAS)

- Main character moves up, down, left and right.
- 5 different DC characters appear with different speeds.
- No difficulty increasing.
- Game over when Marvel unit colides with DC unit.
